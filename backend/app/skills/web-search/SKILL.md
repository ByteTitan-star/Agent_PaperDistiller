# Web Search (Tavily) / 网络搜索

搜索互联网获取实时信息。Search the web for current information, recent research papers, code repositories, or facts not available in the current paper context.

## When to use / 触发条件
- 用户问天气、新闻、实时信息（今天天气怎么样、最近新闻、今天发生了什么）
- User asks about recent/latest information
- Need to compare with current SOTA methods
- Looking for related work not cited in the paper
- 需要查找代码仓库、开源项目、下载链接、GitHub
- Need code repositories or implementation details
- Verifying facts or finding additional resources
- 用户问题超出论文范围，需要外部信息

## Returns / 返回
搜索结果，包含标题、URL 和内容摘要。Top search results with titles, URLs, and content snippets.

# Web Search (Tavily)

Search the web for current information, recent research papers, code repositories, or facts not available in the current paper context.

## When to use
- User asks about recent/latest information
- Need to compare with current SOTA methods
- Looking for related work not cited in the paper
- Need code repositories or implementation details
- Verifying facts or finding additional resources

## Returns
Top search results with titles, URLs, and content snippets.
